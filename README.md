## Tech Stack
- Python
- Flask
- OpenAI API
- SQLite
- HTML/CSS/JavaScript
- Render

  ## Features

- AI-powered incident classification and summarization
- Severity assessment (Low / Medium / High)
- Actionable safety recommendations
- Incident history with status tracking (Open / Under Review / Closed)
- Search and filter functionality
- Dashboard analytics (total incidents, severity distribution, category trends)
- CSV export for reporting

## Architecture

- Frontend: HTML, CSS, JavaScript
- Backend: Flask REST API
- AI Processing: OpenAI API
- Database: SQLite (persistent storage)
- Deployment: Render
